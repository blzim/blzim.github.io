Processors

 

Processors do different things inside of one IC (Integrated Circuit) chip.

They fetch instructions from the Memory Interface with the Instruction Fetcher, decode what instructions need math with the Instruction Decoder, temporarily store the data using Registers, do any needed arithmatic with the ALU (Arithmatic Logic Unit) and move results either back to the Registers for more processing or back out into the system memory.

This system processor cycle is known as the instruction cycle and is often illustrated in Computer Science books using a diagram like the following:

The ALU does math using circuits called "full adders" and "two's compliment".
The full adder circuit adds binary numbers and carries the remainder off to the next adder with a series of boolean logic gates.
(see Digital Logic to learn about binary gates).

Two's Compliment


The two's complement of a binary number takes half the bits and treats them like negative values. Using this method, we can use a Full Adder circuit on negative numbers giving us subtraction. Addition in sets = multiplication and subtraction in sets gives us division, giving a full set of mathematics.
