Operating Systems

An Operating System (OS) is the software that manages a computer. There are two major parts to an OS, the kernel space and the user space. The kernel space is concerned with allowing software and users to work with the hardware.

Kernel functions include:

  • Process Manager = what the CPU is processing.
  • Memory Manager = what the RAM is holding.
  • File Manager = how DATA is being stored.
  • Device Manager = what peripherials are being used.
  • Input Output (I/O) Manager = what is going to and from storage or to and from the user.
  • Network Manager = what is going to and from other systems.
User portion includes:
  • User Manager = who is using the system and what are they allowed to access.
  • Command Line Interface (CLI) or shell = text environment for user to use the system with.
  • Graphic User Interface (GUI) = mouse environment for user to use the system with.
  • System settings = software package managers, system information and status programs, control panel, security tools like anti-virus or firewall programs.
  • Application Programming Interfaces (APIs).
  • Any other pre-installed programs, like word processing, games, programming languages.
