File Types listed by Extention
Extensions are file types that are added to the end of file names. In MS-DOS, an extension could only have up to three letters or digits. Since Windows 95, new file formats generally keep the extension to three characters even though Windows 95, 98, NT and 2000 support long file names and longer extensions. All programs and almost all data files use extensions, which are separated from the file name with a dot. For example, LETTER.DOC is a Word document.
